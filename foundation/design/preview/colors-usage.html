Foundations · Color usage

When to use which color

The palette has three brand anchors and one supporting cream. This page tells you which to reach for, and which combinations meet WCAG AA contrast (4.5:1 for body text, 3:1 for large text and UI).

Heads up. White on orange-500 is only 2.59:1 — it fails AA for any text. The Donate button works because the text is bold and ≥18px (large‑text exception kicks in at 3:1, still fails). For body copy and small UI on orange surfaces, always set foreground to ink-900, not white.

Roles & intent

Each token has a job. Don't pick a color because it looks nice — pick the role.

Ink — voice
--ccpd-ink-900 · #000000
All body type, headings, nav links, default surfaces‑inverse. The default text color, full stop.
Monarch Orange — action
--ccpd-orange-500 · #F98008
Reserved for: primary CTAs, the wordmark sub, link accents, "science fuels hope" emphasis, donate flows.
Blush Pink — warmth
--ccpd-pink-500 · #E8C0C0
Brand surface — closer/CTA backgrounds, hero accent, photo placeholder. Never use for text.
Pink-50 — page
--ccpd-pink-50 · #FDF7F7
Default page background. Replaces white for warmth without competing with brand pink.
Cream-50 — paper
--ccpd-cream-50 · #FBF8F2
Inverse text color on dark surfaces (footer, hero, stats panel). Print/ledger feel.
White — card
--ccpd-white · #FFFFFF
Cards, donation widget, FAQ items. Used to lift content out of the pink page.

Decision shortcuts

When in doubt:

  1. Body text: ink-900. Always. Never tinted.
  2. Muted text (captions, fineprint): gray-500 only on white or pink-50.
  3. Page background: pink-50. Cards on it: white. Don't use white as a section background — it loses warmth.
  4. Primary CTA: orange-500 with ink-900 label. (Repo currently uses white — fails AA.)
  5. Section emphasis: alternate pink-50 → ink-900 → pink-500 → pink-50. Never two darks in a row.
  6. Italic emphasis is the serif font, not just italic Poppins. Use sparingly — once per paragraph at most.
