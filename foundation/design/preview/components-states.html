Focus principles

  1. Every interactive element must have a visible focus state. No exceptions.
  2. Focus rings are 3px solid with 2px offset. Color picks the most readable of: ink-900, orange-500, or orange-400 on dark.
  3. Never rely on browser default outline: blue. It clashes with the brand and is inconsistent across browsers.
  4. Loading states must preserve the button width so the layout doesn't jump. Use a fixed‑width container or visibility: hidden swaps.
  5. Disabled controls must communicate why: empty form, sold out, network error. Pair the disabled state with a helper line.
