# Concert for a Cure — Chasing a Cure Parkinson's Foundation

This repo contains the website and supporting materials for Concert for a Cure 2026.

## Website (Next.js)

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Repo structure

- `app/` — Next.js pages (homepage, concert, about, science, donate)
- `components/` — Shared components (Nav, Footer, DonationWidget, etc.)
- `public/` — Images and assets
- `design/` — Design system and brand guidelines
- `drafts/` — Messaging guides and copy
- `marketing/` — Marketing materials
- `planning/` — Planning documents
- `automation/` — Apps Script and automation tools

## Deploy

Deploy via [Vercel](https://vercel.com). Connect the repo and it deploys automatically on push to `main`.
